IDEAS Café – Website Starter (Free Hosting)

This is a simple, luxury-style, leasing-ready website with 5 pages:
- index.html
- concept.html
- formats.html
- downloads.html
- contact.html
- styles.css

How to host FREE on GitHub Pages:
1) Create a GitHub account.
2) Create a new repository (public) called: ideas-cafe (or any name).
3) Upload all files in this folder to the repository root.
4) Go to Settings → Pages → Source: Deploy from a branch → Branch: main / root.
5) Your website link will appear there.

Optional:
- Add a custom domain later (cost depends on domain provider).
- Replace placeholder download links in downloads.html with your PDF filenames or URLs.
